## 주제: 기상에 따른 선박 닻 끌림 예측
1. 기상 상태에 따른 닻 끌림 발생 여부 분석
2. 현재 시점에서 30분~1시간 후 닻 끌림 발생 예측 모델 아이디어 제안

머신러닝의 분류기법을 이용하여 닻 끌림 예측 시점을 예측한다.

활용 데이터 : 날씨마루에서 제공해준 선박 데이터, 기상 데이터, 해양 데이터
